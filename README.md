## Comandos Git

Esse comando faz:
```cmd
$ git init
```

Esse comando faz:
```cmd
$ git add .
```

Esse comando faz:
```cmd
$ git commit -m "mensagem"
```

Esse comando faz:
```cmd
$ git push
```

pesquisar esses comandos no Git:
```cmd
$ git pull
$ git status
$ git checkout
$ git stash
$ git merg
```
